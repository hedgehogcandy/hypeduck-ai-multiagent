# 📚 레퍼런스 수집 가이드

> 경쟁사 및 레퍼런스 채널을 체계적으로 수집하고 분석하는 방법

---

## 🎯 수집 목표

1. **트렌디한 채널 발굴**: 현재 잘 되는 채널 찾기
2. **성공 패턴 분석**: 왜 잘 되는지 파악
3. **벤치마크 설정**: 우리 계정에 적용할 포인트 추출

---

## 📋 수집 워크플로우

### Step 1: 채널 발굴

```
소스:
1. "회원님을 위한 추천" 피드
2. 알고리즘 피드 추천
3. 해시태그 탐색
4. 릴스 탐색 탭
5. 경쟁사 팔로잉 목록
```

### Step 2: 채널 평가 기준

| 기준 | 최소 요건 |
|------|----------|
| 팔로워 수 | 1만+ (니치는 5천+) |
| 평균 조회수 | 팔로워의 10%+ |
| 댓글 비율 | 조회수 대비 1/30+ |
| 업로드 빈도 | 주 3회+ |
| 성장 속도 | 월 10%+ |

### Step 3: 잘 터진 영상 기준

```
🏆 BEST (10:1 비율)
- 조회수 10만 → 댓글 1만개
- 바이럴 확정, 구조 분석 필수

👍 GOOD (20:1 비율)
- 조회수 10만 → 댓글 5천개
- 좋은 콘텐츠, 참고할 가치 있음

👌 OK (30:1 비율)
- 조회수 10만 → 댓글 3천개
- 평균 이상, 선별적 참고
```

## 📁 저장 구조

```
references/
├── channels/
│   └── {channel_name}/
│       ├── info.json          # 채널 기본 정보
│       ├── metrics.json       # 지표 추적
│       ├── top_videos.json    # 잘 터진 영상 목록
│       └── analysis.md        # 분석 노트
├── videos/
│   └── {video_id}/
│       ├── metadata.json      # 영상 메타데이터
│       ├── script.txt         # 스크립트 추출
│       ├── structure.json     # 구조 분석
│       └── thumbnail.png      # 썸네일
└── trends/
    └── {date}/
        └── daily_report.json  # 일일 트렌드
```
