# 🦆 HypeDuck AI Multi-Agent System

> AI 자가학습 멀티에이전트 시스템 - 계정별 정보 및 학습 데이터 저장소

## 📁 폴더 구조

```
hypeduck-ai-multiagent/
├── accounts/                    # 계정별 정보
│   ├── instagram/
│   │   ├── meow._.king_/       # 계정 1
│   │   │   ├── profile.json    # 프로필 정보
│   │   │   ├── followers.json  # 팔로워 데이터
│   │   │   └── history/        # 활동 히스토리
│   │   └── alligator.92970752/ # 계정 2
│   ├── google/
│   └── naver/
├── learning/                    # 학습 데이터
│   ├── patterns/               # 학습된 패턴
│   ├── models/                 # 모델 설정
│   └── logs/                   # 학습 로그
├── knowledge/                   # 지식 베이스
│   ├── automation/             # 자동화 지식
│   ├── workflows/              # 워크플로우
│   └── templates/              # 템플릿
├── agents/                      # 에이전트 설정
│   ├── chrome-mcp/             # Chrome MCP 설정
│   ├── instagram-bot/          # 인스타 봇 설정
│   └── multi-agent/            # 멀티에이전트 설정
└── config/                      # 전역 설정
    └── settings.json
```

## 🚀 기능

- **계정 관리**: 여러 플랫폼의 계정 정보 저장 및 관리
- **자가 학습**: 작업 패턴 학습 및 자동화
- **멀티에이전트**: 여러 에이전트 협업 시스템
- **지식 축적**: 학습된 내용 영구 저장

## 🔧 사용 기술

- Chrome MCP Server
- Claude AI
- OpenClaw (선택)
- GitHub Actions (자동화)
