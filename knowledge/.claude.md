# 🧠 지식 베이스 가이드

> 학습된 패턴과 인사이트를 축적하는 지식 창고

---

## 🎯 지식 관리 목표

1. **패턴 축적**: 성공/실패 패턴 기록
2. **학습 자동화**: AI가 참조할 수 있는 구조화된 데이터
3. **인사이트 공유**: 계정 간 학습 내용 공유
4. **지속적 개선**: 데이터 기반 의사결정

## 🤖 AI 학습 활용

### Claude에게 제공할 컨텍스트
```markdown
## 이 계정의 학습된 패턴

### 성공 후킹
- 질문형: "이거 알면 인생이 바뀝니다" (성공률 78%)
- 공감형: "다들 이런 경험 있죠?" (성공률 72%)

### 피해야 할 것
- 과장된 충격형 (실패율 65%)
- 너무 긴 도입부 (이탈률 증가)

### 최적 영상 길이
- 30-45초 (완료율 최고)
- 60초 이상 (완료율 급감)
```

### 자동 컨텍스트 생성
```python
def generate_ai_context(account_id):
    patterns = load_patterns(account_id)
    insights = load_recent_insights(account_id)
    
    return f"""
    ## {account_id} 학습 컨텍스트
    
    ### 성공 패턴 Top 3
    {format_top_patterns(patterns['successful'][:3])}
    
    ### 최근 인사이트
    {format_insights(insights)}
    
    ### 권장 사항
    {generate_recommendations(patterns, insights)}
    """
```
