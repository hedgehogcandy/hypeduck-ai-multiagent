# 📝 스크립트 템플릿 가이드

> 잘 터진 영상의 구조를 분석하여 템플릿화하는 방법

---

## 🎯 템플릿 목표

1. **성공 패턴 재현**: 10:1 비율 영상 구조 복제
2. **효율화**: 스크립트 작성 시간 단축
3. **일관성**: 계정별 톤앤매너 유지

---

## 📋 대본 구조 분석

### 기본 구조 (5단계)

```
┌─────────────────────────────────────┐
│ 1. HOOK (0-3초)                     │
│    - 시청자 멈추게 하기             │
│    - 질문, 충격, 공감               │
├─────────────────────────────────────┤
│ 2. PROBLEM (3-10초)                 │
│    - 문제 제기                      │
│    - 공감대 형성                    │
├─────────────────────────────────────┤
│ 3. SOLUTION (10-40초)               │
│    - 해결책 제시                    │
│    - 핵심 내용                      │
├─────────────────────────────────────┤
│ 4. PROOF (40-55초)                  │
│    - 증거/결과                      │
│    - 비포/애프터                    │
├─────────────────────────────────────┤
│ 5. CTA (55-60초)                    │
│    - 행동 유도                      │
│    - 팔로우/저장/댓글               │
└─────────────────────────────────────┘
```

## 📁 템플릿 구조

```
templates/
├── hooks/                    # 후킹 템플릿
│   ├── question.md          # 질문형
│   ├── shock.md             # 충격형
│   ├── empathy.md           # 공감형
│   ├── curiosity.md         # 호기심형
│   └── controversy.md       # 논쟁형
├── structures/              # 구조 템플릿
│   ├── problem-solution.md  # 문제-해결
│   ├── listicle.md          # 리스트형
│   ├── story.md             # 스토리형
│   ├── tutorial.md          # 튜토리얼
│   └── review.md            # 리뷰형
├── ctas/                    # CTA 템플릿
│   ├── follow.md            # 팔로우 유도
│   ├── save.md              # 저장 유도
│   ├── comment.md           # 댓글 유도
│   └── share.md             # 공유 유도
└── styles/                  # 스타일 가이드
    ├── casual.md            # 캐주얼
    ├── professional.md      # 전문적
    └── emotional.md         # 감성적
```

## 🔥 후킹 템플릿 예시

### 질문형
```
"이거 아직도 모르세요?"
"왜 아무도 안 알려줬을까요?"
"00 하고 계신가요? 큰일 납니다"
```

### 충격형
```
"충격적인 사실 알려드림"
"이거 보고 경악했습니다"
"이게 진짜라고요?"
```

### 공감형
```
"저만 이런가요?"
"다들 이런 경험 있죠?"
"이거 공감되면 좋아요"
```
